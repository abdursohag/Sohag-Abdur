# Tutor Tech - Dinajpur

## Agentic Workflow
This project is built using a high-performance, mobile-first agentic workflow:
1. **Architecture Setup**: Initializing Vite + Tailwind 4 + Framer Motion.
2. **Vibe Coding**: Implementing tech-forward UI with glassmorphism and mesh gradients.
3. **Lead Funnels**: Building automated lead generation forms with Zod validation.
4. **Local SEO Authority**: Semantic content clusters for Dinajpur local areas.
5. **Autonomous Testing**: Viewport emulation and lead flow verification.

## Tech Stack
- **Frontend**: React 19, Vite, Tailwind CSS 4
- **Animation**: Framer Motion
- **Forms**: React Hook Form + Zod
- **Icons**: Lucide React
- **SEO**: JSON-LD Local Business Schema
